**outcome** is a lightweight Java library for capturing and analysing test outcomes across multiple runs, enabling trend analysis and regression detection over time.

## Use cases

- Track test result distributions across builds
- Detect gradual performance drift before it becomes critical
- Generate compliance-ready reports showing system behaviour over time
